个人博客搭建

简要概述

  • 环境：win10系统，git配置完成（若未配置请先完成）

Nodejs 安装

  1. 官网下载

    NodeJS-18.5.0直链

  2. 安装完成测试
    node版本查看：node -v

    npm版本查看：npm -v


现有账户与登录账户

hexo 安装

  1. 非科学上网

    cnpm安装：npm install -g cnpm --registry=https://registry.npm.taobao.org

    cnpm版本查看：cnpm -v

    hexo安装：cnpm install -g hexo-cli

    hexo版本查看：hexo -v

  2. 科学上网

    hexo安装：npm install -g hexo-cli

    hexo版本查看：hexo -v

本地博客

  1. 新建文件夹

    新建文件夹 Blog 存放文件，后续操作都在该文件夹，路径一定要记住，如果有做错的地方，删除该文件夹，大不了重新来。

    我的文件夹路径：E:\Blog

  2. hexo 使用

    从 cmd 进入文件夹

    初始化：hexo init 文件夹名称

    启动本地博客服务：hexo s

    hexo d

  3. 本地访问

    本地访问地址

远端博客(github)

  1. github 创建仓库

    在自己的github账户下创建仓库，注意名字 用户名.github.io

  2. 配置 _config.yml

    在 Blog 文件夹下找到 _config.yml 文件，打开后在最后一行添加

  3. 部署

    清理：hexo c

    生成：hexo g

    部署到远程Github仓库：hexo g
